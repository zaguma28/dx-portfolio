# 業務改善・DX開発者 プロフィール

## 長文版（クラウドワークス・ランサーズ プロフィール欄向け）

こんにちは。JIS品質管理者の資格を持ち、現場の業務プロセス改善からIT化・電子化・DX化までを支援している開発者です。

普段はkintoneをメインに使いながら、製造業の品質管理・検査記録・ロット管理・出荷管理・クレーム対応・タスク管理など、現場の業務フローを見える化する開発をしています。

標準機能だけでは扱いにくい複雑な入力、計算、判定、承認、通知を、JavaScript/CSSカスタマイズで整理して、現場が迷わず入力できる画面を作るのが得意です。

最近では生成AIを使ったWebアプリやスマホアプリの制作も行っています。
要件の整理から画面設計、コード生成、テスト観点の抽出まで、AIを活用してスピーディーに進められるのが強みです。

実際の開発では、kintoneアプリ設計・REST API連携・複数アプリのデータ連携・帳票出力・グラフ可視化・エラーログ収集までを一貫して対応しています。品種ごとの合否判定ロジック、在庫位置をマトリクスで管理する画面、ガントチャート、散布図、6ステップのクレーム管理ウィザード、期限リスク付きタスク管理など、現場の判断を支える機能を作ってきました。

「現場の業務を止めない」「間違いに早く気づける」「誰が見ても判断できる」を大切に、あなたの現場のDX化を手伝います。

## 強み

1. **JIS品質管理者としての現場知見**
   品質管理の現場を知った上で、規格・区分・判定基準を正しくアプリに落とし込みます。

2. **kintone標準＋カスタマイズのバランス感**
   標準機能で済む部分は標準で、複雑な部分はJavaScript/CSSで補う、無駄のない設計を心がけます。

3. **現場運用を止めない設計**
   入力補助、バリデーション、エラーログ、自動保存、権限制御など、日々の運用で困らない仕組みを重視します。

4. **生成AIを活用した開発・文書作成**
   Webアプリ・スマホアプリの制作や、要件定義書・設計書・テスト観点の整理を効率的に行います。

5. **業務改善からDX化まで幅広く対応**
   単なるシステム導入ではなく、既存の業務プロセスを改善し、IT化・電子化へつなげる支援ができます。

## 実績ハイライト

- 業務アプリ40以上を開発・カスタマイズ（kintoneを含む）
- 357フィールド、29ファイル・約9,000行の大規模在庫管理アプリを構築
- 多品種の品質判定ロジックを持つ品質データ集約アプリを開発
- 4つのアプリからデータを集約し、最終合否判定を自動化
- 6ステップのクレーム管理ウィザードを設計・実装
- TypeScript + Vite + SendGridを使ったガントチャート型タスク管理アプリを開発
- 生成AIを活用したWebアプリ・スマホアプリの制作経験あり
