# 業務改善・DXポートフォリオ

JIS品質管理者の資格を持ち、業務アプリ開発・JavaScript/CSSカスタマイズ・生成AIを組み合わせた業務改善・DX化の事例です。

## 自己紹介

JIS品質管理者の資格を持ち、現場の業務プロセス改善からIT化・電子化・DX化までを支援している開発者です。

普段はkintoneをメインに使いながら、製造業の品質管理・検査記録・ロット管理・出荷管理・クレーム対応・タスク管理など、現場の業務フローを見える化する開発をしています。

その一方で、生成AIを使ったWebアプリやスマホアプリ、自律型AIエージェントの制作も行っており、要件の整理から画面設計、コード生成、テスト観点の抽出まで、スピーディーに進められます。

「現場の業務を止めない」「間違いに早く気づける」「誰が見ても判断できる」を大切に、標準機能とJavaScript/CSSカスタマイズを使い分けながら、現場が迷わず使えるアプリを設計・開発します。

詳しいプロフィールやクラウドワークス・ランサーズ向けの紹介文は [profile.md](./profile.md) をご覧ください。

## 実績ハイライト

- 業務アプリ40以上を開発・カスタマイズ（kintoneを含む）
- 357フィールド、29ファイル・約9,000行の大規模在庫管理アプリを構築
- 多品種の品質判定ロジックを持つ品質データ集約アプリを開発
- 4つのアプリからデータを集約し、最終合否判定を自動化
- 6ステップのクレーム管理ウィザードを設計・実装
- TypeScript + Vite + SendGridを使ったガントチャート型タスク管理アプリを開発
- 生成AIを活用したWebアプリ・スマホアプリ・AIエージェントの制作経験あり
